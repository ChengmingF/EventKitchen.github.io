# EventKitchen.github.io

Static GitHub Pages project page for EventKitchen.

## Edit the page

- Update project text in `index.html`.
- Replace `assets/images/teaser.jpg` and `assets/video/teaser.mp4` with final media.
- Replace the placeholder author names and affiliations.
- Add the real arXiv, code, and dataset URLs in the three resource links.
- Replace the abstract placeholder with the accepted paper abstract.

## Publish on GitHub Pages

If this project is still under double-blind review, keep the repository private
or remove identifying author/paper links until public release.

This repository is currently named `EventKitchen.github.io` under the
`ChengmingF` account. Unless you move it to an `EventKitchen` GitHub account or
organization, GitHub Pages will normally publish it as a project page:

`https://chengmingf.github.io/EventKitchen.github.io/`

To get a root URL like `https://shape-of-motion.github.io/`, create or use a
GitHub account/organization named `EventKitchen`, then host this code in a repo
named `EventKitchen.github.io`.
